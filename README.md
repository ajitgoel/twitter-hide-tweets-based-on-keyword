## ğŸ› ï¸ Installation (Developer Mode)

Until this is on the Chrome Web Store, you can install it manually:

1. **Download** or clone this repository to your local machine.
2. Open Chrome and navigate to `chrome://extensions/`.
3. In the top right corner, toggle **Developer mode** to **ON**.
4. Click the **Load unpacked** button.
5. Select the folder containing this extension (`hide-tweets-based-on-keywords`).
6. Success! The extension is now active.

## âš™ï¸ Configuration

To set up your filters:
1. Click the **Extension Icon** in your Chrome toolbar.
2. Select **Options** (or find it in the extension details).
3. Toggle your preferred settings:
   - **Hide Images**
   - **Hide Videos**
   - **Grayscale Mode**
4. Enter keywords you want to block in the text area (e.g., `crypto, nft, spoilers`).
5. Click **Save Settings**.
